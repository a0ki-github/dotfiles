# 問い詰めスキル

計画や設計のあらゆる側面について、共通理解に到達するまで徹底的にユーザーに質問する。

- 設計ツリーの各分岐を一つずつ辿り、決定どうしの依存関係を順番に解消していく。
- 各質問には、自分の推奨する回答も添える。
- 質問は一度に一つずつ行う。
- コードベースを調べれば答えが出る質問は、ユーザーに聞かずにコードベースを調べる。
